Lienzo interactivo que genera partículas animadas al mover o hacer clic con el cursor. Incluye cambio de paletas de color y botón de reinicio — todo con Canvas API puro.

## ✨ Live Demo

👉 **[Ver demo en vivo](https://abrahamramoskd.github.io/cursorpro)**
## 🎨 Características

- 🖱️ **Partículas en movimiento** — se generan al mover el cursor por el lienzo
- 🖱️ **Explosión al click** — clic izquierdo crea una ráfaga de partículas
- 🎨 **Cambio de paletas** — alterna entre distintos esquemas de color
- 🔄 **Botón de reinicio** — limpia el lienzo al instante
- ⚡ **Canvas API puro** — sin librerías, solo JavaScript nativo

## 🛠️ Tecnologías

- HTML5 + Canvas API
- CSS puro
- Vanilla JS (requestAnimationFrame, eventos de mouse)

## 📁 Estructura

```
cursorpro/
├── cursorpro.html    ← Aplicación completa

```

## 🚀 Uso

Abre `cursorpro.html` en tu navegador, mueve el cursor por la pantalla y haz clic para ver la explosión de partículas. Usa los botones para cambiar paleta o reiniciar.
